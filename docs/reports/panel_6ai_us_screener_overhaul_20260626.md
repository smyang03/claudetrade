## 2차 — 반박·검증
- **회의론자→전원(핵심 반박):** raw-return 기반 결론 다 의심. **단 인정**: 실현 net에서도 momentum_now −39.7 vs ORP +23.6은 **거래가능 실현치**다 → "급등추격 패배·풀백 승리"는 raw 아닌 **실현으로 확증.** 유동성·hard_cap·$5는 raw뿐이라 **미확증.** → 확실(실현): ORP↑·momentum_now↓·독성격리. 미확증(raw): 유동성·가격·정원.
- **리스크→PM(유동성):** 저유동성 제외군 +0.44%는 **잡으려는 순간 사라질 알파**(저유동=큰 스프레드). raw 전방수익은 mid-price 기준이라 실거래 −비용. **유동성 가중은 건드리지 마라, 슬리피지 실측 전엔.**
- **개발자→회의론자(랭커):** 맞다, raw다. 하지만 **trainer_score 고득점이 실현 패자에 몰림(US15 한계지만 방향 일관)** + 제외>포함 = 랭커가 신호를 못 만든다는 정황 충분. **랭커를 못 믿으면 = 점수로 사이즈업·정원컷 하지 마라**(hard_cap이 승자 자르는 이유).
- **트레이너 보강:** ORP 공급 주3건 제약 인정 → "ORP 강제증량"은 비현실. 대신 **풀백 셋업 정의를 넓혀**(fromHigh 음수·저change) 공급 늘리고, 모멘텀 추격분을 줄여 **상대 비중**을 옮긴다.
- **PM 수렴:** dead weight 43% 컷 = 무비용 효율. 독성종목 격리 = 무논쟁. 유동성·가격은 **슬리피지 실측 후**. 정원(hard_cap)은 **점수 못 믿으니 정원 컷 기준을 점수에서 → 셋업유형(풀백 우선)으로** 바꾸는 게 안전.
- **혁신가 일부 철회:** "전면 풀백 발굴기 전환"은 과격(ORP 공급제약·6월 단일국면). **점진**: 모멘텀 추격분 억제 + 풀백 정의 확대 + 독성격리 + dead weight 컷. 파라미터 동결해제는 **백테스트 없이는 위험**(현 범위가 우연히 안전대일 수도).

### 만장일치 결론
**US 스크리너 = "급등·고유동 추격에 갇혀 풀백 승자를 못 본다"는 구조 편향은 실측 확인.** 단 처방은 증거등급으로 갈린다:
- **즉시(A)**: 독성종목 격리 + dead weight 컷 — 무논쟁·저위험.
- **측정 후(B)**: 모멘텀 억제·풀백 비중 — 이미 켠 게이트와 중복이라 월요일 측정 선결.
- **실측 선결(C/D)**: 유동성·가격·정원·파라미터 — raw-return뿐이라 **슬리피지/백테스트 없이 건들면 위험**(거래불가 알파 함정).

### 냉정한 한 줄
