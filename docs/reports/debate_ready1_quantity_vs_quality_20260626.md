# 토론 리포트 — ready=1 개선 방향: 양(생산↑) vs 질(선별/수익률) (2026-06-26)

> 운영자 질문: ready=1이 ready=0보다 나으면, ready=1을 더 *많이* 만들까, 아니면 *수익률* 관점으로 개선할까.
> 명제: "ready=1 생산량을 늘리는 것(승격률↑)이 net 개선의 우선 방향이다." PRO=양↑ / CON=질(선별).

## 2. 판정 — 명제 **반대 (무차별 증산), 부분 찬성 (US 최상위 한정)**

**"ready=1 무차별 증산(승격률↑·진입기준 완화)"은 틀렸다 (양측 합의):**
- ready=1도 KR 실현 −0.85~0.97% 음수, median 양 시장 음수 = 보통 ready=1도 진다.
- 양 늘리면 한계분이 ready=0(−2~4%) 쪽으로 희석 + 스케일업 함정(net 음수×빈도=손실 확대, 일일캡 PF cap_1 1.40>cap_3 0.69).
- forward 근거는 outlier 가짜(①), 게이트 완화는 −3.57% 코호트 수입(③).

**답은 질 — 단 "질"은 막연한 선별이 아니라 측정된 2갈래:**
1. **수익 코호트 집중 (양·질 일치 지점):** US fired + score≥0.85 + momentum. 여기선 양↑과 질↑이 충돌 안 함 — 이 코호트를 늘리는 건 정당. (단 forward N=28·실현 N=5 소표본, 단일국면.)
2. **실행 회수 (희석 0, PRO 최강):** US fired-ready가 INVALID_PRICE/qty_zero/order_rejected로 버려진 ~14건 — 게이트 안 건드리고 net 순증. **INVALID_PRICE는 watch→buy 토론서 식별한 pathb:4280 fallback 결손과 연결**(단 근본원인=limit_price=0 조사 선행, 나이브 수정 위험).

**KR = 양·질 둘 다 보류** (실현 음수, momentum −0.98%).

## 3. 합의 / 불일치
**합의:** 무차별 증산·게이트 완화·KR 증산·forward 근거 = 전부 틀림. 최상위 코호트(US fired+score≥0.85+momentum)는 양·질 일치라 늘려도 됨. 실행 회수는 희석 0 upside.
**불일치(미미):** "질"의 범위 — PRO는 실행회수 강조, CON은 코호트 라우팅 강조. 둘 다 같은 곳(US 최상위).
**미검증:** 코호트 핵심 증거 N 작음(US momentum 5, score≥0.85 forward 28), 단일국면. **사이즈/증산 결정 전 shadow로 실현 N을 18/14→40+ 누적 필요.**
