# 자동매매 시스템 종합 분석 요청 자료

## 0. 분석 목표

이번 분석의 목표는 다음 4가지를 한 번에 도출하는 것이다.

1. 현재 시스템의 전체 구조, 매매 흐름, 의사결정 흐름, 수익률 현황을 상세히 정리한다.
2. 같은 내용을 바탕으로 현실적이고 부정적인 관점에서 시스템의 약점, 실패 가능성, 구조적 위험을 강하게 비판한다.
3. 같은 내용을 바탕으로 긍정적인 관점에서 시스템의 강점, 이미 작동 중인 부분, 앞으로 키울 만한 가능성을 정리한다.
4. 위 3가지를 종합해서 지금부터 우선적으로 개선해야 할 방향과 추가 설계 포인트를 도출한다.

## 8. 특별히 지켜야 할 기준

- 수익률 평가는 KR/US를 반드시 분리한다.
- PathA와 PathB를 섞어서 평가하지 않는다.
- selection 문제와 execution/risk 문제를 섞지 않는다.
- 단기 성과만 보고 전략 철학을 바꾸지 않는다.
- US PathB claude_price 수익 엔진은 특별 보호 대상으로 본다.
- broker truth, 주문 안전장치, hard stop, profit ladder, hold advisor 관련 변경 제안은 신중하게 다룬다.
- 개선안은 기본적으로 live/enforce 적용 가능성을 전제로 설계한다.
- shadow는 정말 불확실한 경우에만 제안한다.
