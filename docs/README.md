# ClaudeTrade Docs

This folder is now intentionally small.

Start here:

- [important/README.md](important/README.md) - curated document map and title summaries.
- [important/ACTIVE_WORK.md](important/ACTIVE_WORK.md) - current work that still needs action.
- [important/ALWAYS_ANALYZE.md](important/ALWAYS_ANALYZE.md) - recurring checks before market, live, and policy decisions.
- [important/CLEANUP_REPORT_20260522.md](important/CLEANUP_REPORT_20260522.md) - what changed during the document cleanup.

Structure:

- `important/core/` keeps enduring reference docs.
- `important/source/` keeps source reports that still affect current decisions.
- `archive/` keeps old devlogs only.

Removed by policy:

- `docs/plans/` one-off plans after their remaining action was absorbed.
- `docs/reports/` raw dated reports, simulations, QA outputs, and JSON artifacts after important lessons were summarized.
